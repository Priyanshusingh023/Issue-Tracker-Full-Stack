# Full Stack Issue Tracker System (MERN)

## Overview

The **Issue Tracker System** is a full-stack web application developed using the MERN stack that allows users to create, manage, and track issues efficiently.
This project demonstrates complete end-to-end development including authentication, backend API design, database integration, and responsive frontend UI.

## Key Features

* User Registration and Login with JWT Authentication
* Secure Password Hashing using bcrypt
* Create, Update, Delete Issues
* Issue Status Management (Open / Closed)
* Dashboard to view and manage all issues
* RESTful API integration between frontend and backend
* Responsive UI built with React

## Tech Stack

**Frontend**

* React.js
* CSS / Tailwind CSS
* Axios

**Backend**

* Node.js
* Express.js
* MongoDB (Mongoose)
* JWT Authentication
* bcrypt.js
